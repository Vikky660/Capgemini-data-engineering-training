# Capgemini Data Engineering Training Journey

This repository documents my learning progress and practical work completed during the Capgemini Data Engineering Training Program. The focus of this training is to build strong foundations in data engineering concepts through structured learning and hands-on implementation.

## Objective
The objective of this training is to develop industry-ready skills in Data Engineering by learning SQL, PySpark, and modern data processing techniques through continuous practice and real-world problem solving.

## Week 0 Overview
Week 0 concentrates on establishing core fundamentals required for data engineering:

- SQL fundamentals and query writing
- Introduction to PySpark
- Basic data processing concepts
- Understanding data workflows

## Learning Outcomes
During this phase, I learned:

- Writing SQL queries using SELECT, WHERE, GROUP BY, and JOINS
- Translating SQL operations into PySpark transformations
- Performing data transformations using PySpark DataFrames
- Understanding basic data pipeline concepts
- Working with structured datasets

## Learning Approach
The training was approached with a practical mindset by:

- Practicing queries regularly
- Focusing on understanding logic instead of memorization
- Implementing concepts using PySpark
- Verifying outputs and analyzing results
- Maintaining structured documentation of progress

## Repository Structure
- week0 → foundational learning
- upcoming weeks → advanced concepts and projects

## Goal
The long-term goal of this training journey is to become a skilled Data Engineer capable of designing, building, and maintaining scalable data pipelines and solving real-world business data challenges.
