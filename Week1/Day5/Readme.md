# 📊 Week 1 – Day 5: Advanced SQL (String, Numeric & REGEX Analysis)

---

## 🚀 Objective

To build strong SQL fundamentals by solving real-world problems using:

- String Functions  
- Numeric Functions  
- Date Functions  
- CASE Statements  
- REGEX (Regular Expressions)  

This assignment focuses on applying SQL transformations and pattern extraction techniques to structured datasets.

---

## 🧩 Problem Overview

This project consists of two major parts:

### 1️⃣ Structured Data Analysis (String + Numeric + Date)
- Employee Data  
- Orders Data  
- Customer Spending  
- Subscriptions  
- Loans  
- Attendance  
- Product Sales  
- Insurance  
- Salary & Banking Data  

### 2️⃣ REGEX-Based Data Extraction
- Email parsing  
- Phone number validation  
- Pattern extraction from mixed strings  
- Text manipulation  

---

## 🎯 Goals

- Apply string transformations (UPPER, LOWER, INITCAP)  
- Perform numeric calculations (ROUND, ABS, POWER, MOD)  
- Handle NULL values safely  
- Work with date functions (DATEDIFF, YEAR, MONTH, DAY)  
- Use CASE statements for classification  
- Extract patterns using REGEX  
- Solve real-world business problems using SQL
